Quick start

Quick start guide for using NGxTW in your Angular project.

Prerequisites

NGxTW require Tailwind CSS to apply style to components properly, if it not yet installed in you project, please follow this link to install Tailwind CSS.

Install ngxtw from npm npm i ngxtw Update Tailwind CSS config file

Update the tailwind.config.js file generated by Tailwind CSS at the root of your project.

Install Tailwind CSS forms plugin

This is required by forms components. Ignore this step if you will not use them.

Install the forms plugin from npm

{{'npm install -D @tailwindcss/forms'}}

Update the tailwind.config.js file

What's next?

Go to the components documentation to see how to use each component.
