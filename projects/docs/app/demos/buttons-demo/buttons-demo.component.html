Buttons

Use these Tailwind CSS button components for Angular to surface the different actions someone can perform in your interface.

Primary buttons

Primary buttons have the most visual impact after the FAB, and should be used for important, final actions that complete a flow, like Save, Button text, or Confirm.

Tonal buttons

A tonal button is an alternative middle ground between primary and secondary buttons. They’re useful in contexts where a lower-priority button requires slightly more emphasis than a secondary would give, such as "Next" in an onboarding flow.

Secondary buttons

Secondary buttons are medium-emphasis buttons. They contain actions that are important, but aren’t the primary action in an app.

Text buttons

Text buttons are used for the lowest priority actions, especially when presenting multiple options.

Icon buttons

Icon buttons help people take minor actions with one tap

Icon buttons with text

Icon buttons with text help people take minor actions with one tap

FAB (Floating Button text Buttons)

Floating action buttons help people take primary actions

A reference to the class list instance of the button.

A reference to the button's native element inside of a view.

The button size.

The button variant.
