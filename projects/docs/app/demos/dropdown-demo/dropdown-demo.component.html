Properties Description
{{"@Input({"}}
{{"transform: stringToArray,"}}
{{"}) class: ClassName[] = []"}}

The initial class list setted before the dropdown initialization.
{{"classList: ClassList = new ClassList()"}}

A reference to the class list instance of the dropdown.
{{'nativeElement: HTMLElement'}}

A reference to the native element of the dropdown inside of a view.
{{"@Input({ transform: booleanAttribute })"}}
disabled: boolean = false

Whether the dropdown is disabled.
{{"@Input({ transform: booleanAttribute })"}}
opened: boolean = false

Whether the dropdown is opened.
Methods Description
disable(): void

Disables the dropdown (making it non-interactive).

enable(): void

Enables the dropdown (making it interactive).

open(): void

Opens the dropdown.

close(): void

Closes the dropdown.

toggle(): void

Toggles the dropdown.

setPosition(position: OverlayPosition): void

Sets the position of the dropdown.

closeAfter(delay: number): void

Closes the dropdown after the specified delay.

Events Description
{{"@Output('open') onOpen: EventEmitter\"}}

Event emitted when the dropdown is opened.

{{"@Output('close') onClose: EventEmitter\"}}

Event emitted when the dropdown is closed.

{{"@Output('toggle') onToggle: EventEmitter\"}}

An observable that emits the state (open/closed) of the dropdown when it is toggled.
