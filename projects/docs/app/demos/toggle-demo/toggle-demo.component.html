Toggles

Use these Tailwind CSS toggle components for Angular to add on/off switches for settings like dark mode, notifications, and feature flags.

Properties Description
{{"@Input({"}}
{{"transform: stringToArray,"}}
{{"}) class: ClassName[] = []"}}

The initial class list setted before the toggle initialization.
{{"classList: ClassList = new ClassList()"}}

A reference to the class list instance of the toggle.
{{'nativeElement: HTMLElement'}}

A reference to the native element of the toggle inside of a view.
{{"@Input({ transform: booleanAttribute })"}}
disabled: boolean = false

Whether the toggle is disabled.
{{"@Input({ transform: booleanAttribute })"}}
checked: boolean = false

Whether the toggle is checked.
{{"@Input() slider: ClassName[] = []"}}

The initial class list of the toggle's slider setted before initialization.
{{'checkbox: ElementRef\'}}

A reference to the native checkbox input element wrapped by the toggle.
{{"sliderClassList: ClassList = new ClassList()"}}

A reference to the class list instance of the toggle's slider.
Methods Description
disable(): void

Disable the toggle (making it non-interactive).

enable(): void

Enables the toggle (making it interactive).

toggle(): void

Toggles the toggle.

Events Description
{{"@Output('toggle') onToggle: EventEmitter\"}}

An observable that emits the state of the toggle when it changes.
