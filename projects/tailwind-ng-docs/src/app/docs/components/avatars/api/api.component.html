Methods
Inherited methods
Name Description
{{"focus(options?: FocusOptions): HTMLElement | undefined"}} Focuses an element and return it. If it cannot be focused, nothing is done and undefined is returned.
{{"setVisualfocus(options: FocusOptions): HTMLElement | undefined"}} Sets visual focus on an element and return it. If it cannot be visual focused, nothing is done and undefined is returned. Visual focus is set by adding the data-visual-focused attribute on that element.
{{"removeVisualfocus(target?: HTMLElement): void"}} Removes visual focus on the target element. Default target is the component's native element.
{{"scrollIntoView(options: ScrollIntoViewOptions): void"}} Scrolls the element into the view if needed. Making it visible to the user.
