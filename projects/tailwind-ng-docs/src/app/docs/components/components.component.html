Components overview

Tailwind NG provides a set of beautiful, fast and accessible UI components crafted with Tailwind CSS and Angular, to make building modern web applications easier and faster at any scale.

These components are designed to be highly customizable and easy to use. While some components are still in development, there are many that are ready to use in production. The library is constantly updated with new components and features.
