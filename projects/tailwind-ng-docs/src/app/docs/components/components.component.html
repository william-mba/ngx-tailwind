Components overview

Tailwind NG provides a set of beautiful, fast and accessible UI components crafted with Tailwind CSS and Angular.

These components are designed to be highly customizable and easy to use. While some components are still in development, there are many that are ready to use in production. The library is constantly updated with new components and features.

Avatars

To display user profile pictures and company logos individually or grouped in stacks.

Badges

To highlight important information, statuses, or counts in your application.

Buttons

To represent the different actions users can perform.

Button Groups

To group buttons with other elements like dropdowns and checkboxes, for things like filtering options, pagination, or buttons with multiple actions.

Checkboxes

To add multi-option selection to your lists and setting screens.

Comboboxes

To provide a combination of input fields and dropdown menus for things like autocomplete search bars, country selection, and filtering options.

Modal Dialogs

To create pop-up boxes for things like alerts, notifications, or confirmation prompts.

Dropdowns

To create dropdown menus for action items and navigation, featuring section dividers, icons, and headers.

Icon Buttons

To represent actions, objects, and concepts in a visual way.

Input Groups

To create things like search bars with buttons, credit card forms, and other form inputs with combined input and label elements.

Radio Groups

To create options for things like settings, product variants, and plan comparisons.

Toggles

To add on/off switches for settings like dark mode, notifications, and feature flags.
